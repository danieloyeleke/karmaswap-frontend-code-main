# KarmaSwap Frontend

**Subtitle:** A peer-to-peer marketplace where users trade items using karma currency with escrow protection.

## Features

- **Marketplace**: Browse and search items by category
- **Escrow-Protected Trades**: Secure buyer/seller transactions with karma lockdown
- **Seller Dashboard**: Manage listings, track orders, handle disputes
- **Order Tracking**: Real-time status updates (ESCROW → DISPATCHED → DELIVERED → COMPLETED)
- **Social System**: Follow sellers, view seller ratings
- **Daily Rewards**: Karma earn mechanism
- **Admin Panel**: User/escrow/flag management
- **Notifications**: Real-time trade updates and alerts
- **Authentication**: Google OAuth + Supabase integration

## Tech Stack

- React 18 + Vite
- React Router v7
- Axios for API calls
- Supabase (auth/data)
- STOMP + SockJS (WebSockets)
- Lucide React (icons)

## Getting Started

### Prerequisites
- Node.js 16+
- npm or yarn

### Installation

npm install
npm run dev
